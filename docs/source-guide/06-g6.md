# 06 · 通用功能 / Lite 特性 / 平台

> backend/app/features · workspace / memory / skills / workflows / lite / admin / auth / impersonation

**workspace.service** 的路径沙箱支撑所有文件操作；**skills.loader.parse_frontmatter** 被 skills 与 workflows 复用；**infra.llm.light.json_call** 是全部 lite 特性(triage/gate/ranker/insight)+PDF/insight 增强的统一轻模型入口(全 fail-open)；**capabilities.auth_context** 的 current_user/AuthContext 把 auth、admin、impersonation 串成权限审计体系。

!!! note "建议阅读顺序"
    workspace(service→tool→router→pdf_index) → memory(tool→router) → skills(loader→catalog→tool→freshness) → workflows(registry→service→tool) → lite(triage/skill_gate/memory_ranker/spawn_advisor) → auto_research → auth(util→router→invites) → admin → impersonation

### `backend/app/features/workspace/service.py`

**优先级：** 🔴 核心　·　workspace 路径沙箱与 session→物理路径解析的地基。

- `_path_cache` workspace_id→fs_path(启动由 DB 填)；new_workspace_root 用 <root>/<user>/<ws> 布局
- `resolve_safe_under_root`：agent 路径当 workspace 相对解析并校验不越界，越界抛 PathDenied

**依赖：** config.WORKSPACE_ROOT

### `backend/app/features/workspace/tool.py`

**优先级：** 🔴 核心　·　暴露给 agent 的文件工具 read/write/edit/list_dir，约束在 workspace root。

- `read_file` 多态：按后缀分 dir/pdf/docx/zip/image(vision caption)/text；文本带行号+行/字节双截断
- PDF 默认走 sidecar 索引摘要(pdf_index)缺失则原始页+后台 schedule_index
- `write_file` 冲突默认追加 -1/-2(renamed)；`edit_file` 锚点精确替换(要求唯一匹配)

**依赖：** repo、workspace.service、pdf_index、infra.llm.vision

### `backend/app/features/workspace/router.py + pdf_index.py + pdf_upload_hook.py`

**优先级：** 🟡 重要　·　workspace REST + PDF sidecar 索引 + 上传事件钩子。

- router：tree/file/raw(预览)/download(文件或目录 zip)/mkdir/upload；多文件上传 publish FolderUploaded
- pdf_index：每 (路径,hash) 仅 lite 蒸馏一次，sidecar 落 .idx/，幂等读或构建
- pdf_upload_hook：订阅 FolderUploaded 对 .pdf 预调度索引

**依赖：** repo、auth、workspace.service、infra.llm.light、event_bus

### `lite 特性：triage_guard / skill_gate / memory_ranker / spawn_advisor`

**优先级：** 🟡 重要　·　四个轻模型特性，统一经 light.json_call(op='lite.*')，全 fail-open。

- `triage_guard`：triage_user_message(malicious + simple_reply) + advise_spawn(recommend+preset)
- `skill_gate`：select_relevant_skills 预加载 ≤2 skill(偏向预加载，误报仅 ~3k token)
- `memory_ranker`：select_relevant_memories(≤5) + 本地 conf×30 天半衰期重排
- `spawn_advisor`：enhance_insight 为 insight 生成 title+场景 summary

**依赖：** infra.llm.light.json_call、repo、lib.clock

### `backend/app/features/auth/* + admin/router.py + impersonation/*`

**优先级：** 🟡 重要　·　认证(PBKDF2+session cookie+邀请) + admin CRUD + 借身份审计。

- auth.util：hash_password(PBKDF2-HMAC-SHA256 200k 迭代)、gen_invite_code、gen_session_token
- auth.router：signup(校验邀请)/login/logout/me(含 actingAs)，httponly cookie + bearer
- admin.router：workspaces/sessions/messages 大型 CRUD，按 turn 分页，prod 剥离 tool 原始内容；per-user LLM 设置(从不回传 key)
- impersonation：mint/lookup/revoke 令牌(KV TTL=15min) + record_audit(审计环 ~5k 条)

**依赖：** auth.util、capabilities.auth_context、infra.state、repo
