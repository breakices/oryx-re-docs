# 09 · 前端组件与 lib 工具

> src · components / lib

三栏布局(Sidebar/ChatPane/Workspace)。**四大件**：AppShell(骨架)、ChatPane(数据流中枢)、Message(渲染核心)、Workspace(右栏+金融工作台 ~2500 行)；配套必读 **lib/merge.ts**(后端一行一角色 → 每 turn 一气泡) 与 **lib/markdown.ts**(流式渲染稳定化)。

!!! note "建议阅读顺序"
    AppShell → lib/merge → ChatPane → Message → lib/markdown → ToolStream → Composer → Workspace → Sidebar → lib/async+dialog+DialogHost → 小 lib → 认证组 → 后台组 → 导出/划词组 → markdown 渲染细节 → 金融外围 → 基础叶子件
