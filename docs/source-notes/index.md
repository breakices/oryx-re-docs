# 源码逐文件笔记

每个源码文件对应一个 md，已自动抽出该文件的**类/函数/关键符号清单（带行号）**，你打开源码对照着逐个写。

- 共 **272** 个文件（`backend/app/**/*.py` + `src/**/*.{ts,tsx}`，排除自动生成的 `schema.gen.ts`）。
- 想先了解文件大致职责，去 [源码导读](../source-guide/index.md) 看对应模块卡片。
- 符号清单由轻量正则抽取，**仅作锚点**，可能漏掉动态定义；以源码为准。
- 重复运行脚本只补建缺失文件，**不覆盖你已写的内容**。
